Welcome to our Membership Policies Page.


Hello Coders,

If you enjoy coding or are looking to get into it, Casa De Code Programming Club has an excellent opportunity for you.

The club is open to all students. The only requirement is a keen interest in computer programming.

To allow an equal opportunity for all students to join a club, if your student signs up for a club he/she must attend. Students cannot sign up for multiple clubs that have more than one conflicting date.

All students must be picked up on time when the club is over. If a student is picked up late more than one time, they may be removed from the club. Students are expected to use the manors they are taught to use in the classroom. Misbehavior may result in removal from the club.

Location : Virtual

Participants: Any students only

Website: club.casadecode.com

Requirements / Important info:

  • Hosted by the Casa De Code Computer Programming Club
  • Students do not need to have any prior Coding Experience to participate
  • Students will be able to communicate with each other via discord
