Casa de code Programming Club is a computer programming club.learn more about Casa De Code Programming Club in this Document. The club was established in 2019.The Computer Programming Club have Committee.This club is run by them.Every year a new committee is formed based on the views of the members. The committee has various memberships.If you a member of Casa De Code Programming Club, You can take any one membership if you want.Read this Document about Committee Formation Process. Since its inception, the club has been aiming to encourage young people to do programming.

Programming has gradually become popular in Bangladesh. Competitions, hackathons, etc. are being held occasionally at various levels around programming. In the meantime, some are again organizing various clubs on their own initiatives and organizing various events to interest students in programming there. This is one such computer programming club.

If you are interested in programming, you can join this club. Read Membership Policies of Casa De Code Programming Club in this Document .
